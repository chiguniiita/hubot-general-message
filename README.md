# hubot-general-message

A hubot script that messaging httprequest result

See [`src/general-message.coffee`](src/general-message.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-general-message --save`

Then add **hubot-general-message** to your `external-scripts.json`:

```json
[
  "hubot-general-message"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
